# ののかの推し活ルーム — 設計仕様

## 概要

六本木のレンタルスペース（DEARROOM六本木）への集客を目的とした推し活ブログ。スペースオーナーのキャラクター「ののか（のの）」が推し活の魅力を発信する。自然検索（SEO）経由での流入を狙う。

## キャラクター設定

- **名前**：ののか（愛称：のの）
- **設定**：推し活大好きで六本木にシネマルームを作っちゃったゆるかわ女子
- **口調**：基本はですます形式。テンションが上がるとゆるかわな話し方が出る

## サイト構成

### ナビゲーション（4項目）

| ラベル | パス |
|--------|------|
| ホーム | `/` |
| ブログ | `/blog` |
| スペース紹介 | `/space` |
| プロフィール | `/profile` |

### ページ一覧

#### トップページ（`/`）
- ヒーローエリア：ののかの自己紹介文（ゆるかわキャラ口調）＋デコレーション
- 最新ブログ記事カード（3枚、グリッド表示）

#### ブログ一覧（`/blog`）
- カード型グリッド（モバイル1列・PC2列）
- 記事タイトル・日付・サムネイル表示

#### スペース紹介（`/space`）※新規作成
- DEARROOM六本木のシネマルーム紹介（仮テキスト）
- 予約リンクボタン（URLプレースホルダー：`#`）

#### プロフィール（`/profile`）※新規作成
- ののかのキャラクター紹介（仮テキスト）

#### ブログ記事詳細（`/blog/[slug]`）
- 既存レイアウト流用、フォント・カラーのみ更新

## 削除・整理

- 既存サンプル記事5本（`first-post.md`等）を削除し、上記3本に差し替え
- `src/pages/about.astro` → `src/pages/profile.astro` に差し替え（`/about` は削除）
- `src/pages/space.astro` を新規作成
- Atkinsonフォントファイル（woffファイル）は残す、`astro.config.mjs`のフォント設定のみ削除
- SNS関連リンクはすべて削除
- `src/consts.ts`：`SITE_TITLE` と `SITE_DESCRIPTION` を更新（RSS・メタタグに波及）
